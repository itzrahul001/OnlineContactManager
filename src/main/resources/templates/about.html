About Online Contact Manager

Efficiently manage your contacts with ease and security

Overview

The Online Contact Manager is a robust Spring Boot application designed to simplify the process of managing personal and professional contacts. Built with a focus on security and user experience, this application incorporates key technologies like Spring Security and Thymeleaf for seamless functionality and a responsive UI.

Features

User Authentication and Security

Implemented using Spring Security, ensuring secure login and protection against unauthorized access.

Contact Management
  • Add new contacts easily with a user-friendly form.
  • Update existing contact details as required.
  • View and manage all your contacts in one place.
Responsive Design

Built using Thymeleaf and inline CSS for a clean and mobile-friendly interface.

Technologies Used

  • Spring Boot: For creating a robust and scalable application backend.
  • Spring Security: To handle user authentication and authorization securely.
  • Thymeleaf: For dynamic server-side rendering of HTML templates.
  • Hibernate: For database operations and ORM functionality.

Future Enhancements

The project aims to include additional features such as:

  • Role-based access control for advanced user management.
  • Integration with third-party APIs for importing/exporting contacts.
  • Enhanced search functionality with filters.
